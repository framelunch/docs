## 書き始める
### **new** して
`docs new -i ${DOCUMENT_ID} -w ${WORK_DIR} -b ${BUILD_DIR}`
### **post** して
`docs post -p _index.md`

___
## 見る
`docs preview`

___
## ビルド
`docs build`

___
## 別Documentに切り替える
`docs switch -i ${DOCUMENT_ID}`
